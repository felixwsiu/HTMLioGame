## Technologies
Project is created with:
* Javascript
* Node.js version 10.15.1
* NPM version 6.4.1

	
## Setup
To run this project, 

```
$ npm i express bcryptjs passport passport-local ejs express-ejs-layouts mongoose connect-flash express-session socket.io
$ npm run dev
```
